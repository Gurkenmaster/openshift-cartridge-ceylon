An IEEE 754 64-bit floating point number. A Float is capable of approximately representing numeric values between 2-1022 and (2-2-52)Ã—21023, along with the special values infinity and -infinity, and undefined values (Not a Number). Zero is represented by distinct instances +0, -0, but these instances are equal. An undefined value is not equal to any other value, not even to itself.
